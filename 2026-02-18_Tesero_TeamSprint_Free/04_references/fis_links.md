## Notes on FIS Code Data

Several athlete profiles record a **competitor ID** (the numeric ID used in FIS database URLs) but list the **FIS Code** field as [DATA NOT FOUND]. The competitor ID is functionally equivalent for building profile URLs and is what has been used in the table above for those athletes. The FIS Code is a separate formatted identifier (e.g., 3425499 for NOR, 3505800 for SWE) that may differ from the competitor ID.
